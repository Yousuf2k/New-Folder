What is DOM

DOM stands for Document Object Model. It's a programming interface that represents an HTML document as a tree structure. This structure consists of nodes, which can be elements, attributes, text, or comments.

How to select elemnt from dom

To select elements from the DOM in JavaScript, you can use the following methods: getElementById(): Selects an element by its ID. getElementsByClassName(): Selects elements by their class name. getElementsByTagName(): Selects elements by their tag name. querySelector(): Selects a single element using a CSS selector. What is event delegation in Dom

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future

What is Js

JavaScript, often abbreviated as JS, is a programming language and core technology of the Web, alongside HTML and CSS. 99% of websites use JavaScript on the client side for webpage behavior.
